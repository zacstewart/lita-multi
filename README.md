# lita-multi

Enables [Lita][1] to connect to multiple chat services using different
adapters.

## Installation

Add this line to your application's Gemfile:

    gem 'lita-multi'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install lita-multi
